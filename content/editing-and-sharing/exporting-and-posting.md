# Exporting and Posting
After editing a photograph, I still need to prepare it for posting. I check the crop, image quality, caption, and overall appearance before uploading it. Taking a few extra minutes at this stage helps prevent mistakes such as cutting off the truck or posting the wrong version.

## Save the Final Version

I make sure the edited photograph is saved at full quality. I keep the original image so I can return to it later if I want to create a different edit. I can also place finished photographs in a separate album labeled for Instagram or truck photography.

## Check the Crop

Social media may display photographs differently depending on whether they are horizontal, vertical, or square. Before posting, I preview the crop and confirm that the tires, bumpers, mirrors, and other important parts remain visible.

Vertical photographs can take up more screen space, but I do not force every picture into a crop that cuts off the truck.

## Write a Simple Caption

The caption can describe the location, time of day, recent change to the truck, or something I liked about the picture. It does not need to be extremely long. I try to write something that sounds natural instead of using the same caption for every post.

## Use Relevant Hashtags

I can include a small number of hashtags related to the truck, photography, location, or vehicle style. Using too many unrelated hashtags may make the post look cluttered.

## Final Review

Before pressing the post button, I review the picture, caption, crop, and account selection. I also check whether the photograph fits with the style described on the [[editing-and-sharing/creating-a-consistent-feed|Creating a Consistent Feed]] page.

## Posting Checklist

- [ ] Save the final edited version.
- [ ] Keep the original photograph.
- [ ] Check the image crop.
- [ ] Confirm the truck is not cut off.
- [ ] Write a natural caption.
- [ ] Add a few relevant hashtags.
- [ ] Proofread everything.
- [ ] Preview the post before publishing.
